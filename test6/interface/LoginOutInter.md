# 接口: loginOut | [返回](../Markdown/LoginOut.md)
用例：登出

* 功能：退出登录
* 权限：已登录的用户
* API请求地址：接口基本地址/v1/api/loginOut
* 请求方式：POST
* 请求实例:
```json
{
  "user_id": "001"
}
```
* 请求参数说明：

| 参数名称 |     说明     |
| :------: | :----------: |
| user_id  |    用户id    |

* 返回实例：
```json
{
  "status": "true",
  "info":""
}
```
* 返回参数说明:

| 参数名称 |                          说明                           |
| :------: | :-----------------------------------------------------: |
|  status  | 用户是否可以退出登录，true为允许登出，false为不允许登出 |
|   info   |              额外的信息，可以存放错误信息               |
